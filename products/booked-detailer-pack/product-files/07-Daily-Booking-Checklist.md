# Daily Booking Checklist

Use this at the start and end of each workday.

## Morning
- [ ] Reply to all overnight inquiries
- [ ] Follow up on yesterday’s unclosed quotes
- [ ] Confirm today’s bookings
- [ ] Check available slots before quoting

## During the day
- [ ] Ask for car model, service, and photos before quoting
- [ ] Give one recommended package first
- [ ] End each chat with a clear next step
- [ ] Log serious leads in the tracker

## Evening
- [ ] Follow up on today’s open quotes
- [ ] Confirm tomorrow’s bookings
- [ ] Message 1 to 3 old leads or past customers
- [ ] Note common objections or questions for improvement
